# GhostBrowser Resource Fixtures

This repository hosts importable resource files for GhostBrowser.

## Subscription Center Test Data

- `ghost-subscription-all-resources.json`: a `ghost.subscription.bundle` fixture that covers all subscription center resource kinds: site, lexicon, radar keyword, and radar source.

Use the GitHub raw URL as a subscription URL in GhostBrowser and choose the `全部资源` scope.
